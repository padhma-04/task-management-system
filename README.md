# 🗂 Task Management System

![GitHub repo size](https://img.shields.io/github/repo-size/padhma-04/task-management-system)
![GitHub stars](https://img.shields.io/github/stars/padhma-04/task-management-system?style=social)

A modern **full-stack Task Management System** built using **Django (Backend)** and **React (Frontend)** with a clean UI and dark mode support.

---

## 🚀 Features

* ✅ Add, delete, and update tasks
* ✔ Mark tasks as completed / undo
* 🌙 Dark / Light mode toggle
* ⚡ Fast API integration using Axios
* 🎨 Clean and responsive UI
* 🔄 Real-time updates

---

## 🛠 Tech Stack

**Frontend:**

* React.js
* Axios
* CSS (Custom styling)

**Backend:**

* Django
* Django REST Framework

**Database:**

* SQLite

## ⚙️ Installation & Setup

### 🔹 Clone the repository

```bash
git clone https://github.com/padhma-04/task-management-system.git
cd task-management-system
```

---

### 🔹 Backend Setup (Django)

```bash
cd backend_django/backend
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

---

### 🔹 Frontend Setup (React)

```bash
cd frontend
npm install
npm start
```

---
